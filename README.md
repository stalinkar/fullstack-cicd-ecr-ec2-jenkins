# Fullstack CI/CD with Jenkins, AWS ECR, and EC2

This project demonstrates a complete CI/CD pipeline for a frontend + backend application using:

- GitHub
- Jenkins Pipeline
- Docker
- AWS ECR
- AWS EC2
- SSH-based deployment

## Components
- Frontend application
- Backend application
- Dockerfiles for both services
- docker-compose deployment
- Jenkinsfile for CI/CD automation

## Target Flow
GitHub -> Jenkins -> Docker Build -> AWS ECR -> EC2 Deployment
